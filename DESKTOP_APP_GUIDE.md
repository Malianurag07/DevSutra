This guide explains how to use DevSutra as a standalone application on your Windows laptop.

## üöÄ One-Click Launcher

We have included a smart launcher script: **`run_devsutra.bat`**.

**How to use:**
1.  Open the `DevSutra` folder.
2.  Double-click `run_devsutra.bat`.
3.  **Wait.** The first run might take 2-3 minutes to install everything.
4.  The app will open automatically in your browser!

## ‚öôÔ∏è Prerequisites (Install Once)

Since this runs on your laptop, you need two standard tools installed:

1.  **Python** (for the backend logic)
    *   Download: [python.org](https://www.python.org/downloads/)
    *   **IMPORTANT**: Check the box **"Add Python to PATH"** during installation.

2.  **Node.js** (for the frontend interface)
    *   Download: [nodejs.org](https://nodejs.org/) (Download "LTS" version)
    *   Just click Next -> Next -> Finish.

## üé® Professional Touch: Create a Desktop Shortcut

Want it to feel like a real app?

1.  Right-click `run_devsutra.bat`.
2.  Select **Send to** -> **Desktop (create shortcut)**.
3.  Go to your Desktop.
4.  Right-click the new shortcut -> **Properties**.
5.  **Change Icon**: You can download any `.ico` file you like and set it here.
6.  **Rename**: Rename the shortcut to "DevSutra".

Now you have a "DevSutra" icon on your desktop that launches your app!
